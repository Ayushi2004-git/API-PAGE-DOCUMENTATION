API Endpoints

Endpoints

  • 1. GET /api/resource
  • 2. POST /api/resource
  • 3. DELETE /api/resource/{id}
  • 4. PUT /api/resource/{id}

1. GET /api/resource

Retrieve a list of resources.

Request

GET /api/resource

Response

{
                "data": [
                    {
                        "id": 1,
                        "name": "Resource 1"
                    },
                    {
                        "id": 2,
                        "name": "Resource 2"
                    }
                ]
            }

2. POST /api/resource

Create a new resource.

Request

POST /api/resource
Content-Type: application/json

{
    "name": "New Resource"
}

Response

{
    "id": 3,
    "name": "New Resource"
}

3. DELETE /api/resource/{id}

Delete a resource by ID.

Request

DELETE /api/resource/1

Response

{
            "message": "Resource deleted successfully."
        }

4. PUT /api/resource/{id}

Update an existing resource.

Request

PUT /api/resource/1
Content-Type: application/json

{
    "name": "Updated Resource"
}

Response

{
    "id": 1,
    "name": "Updated Resource"
}
